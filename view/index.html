Welcome to the Blockchain Network

Blockchain is a distributed ledger technology is which nodes communicate with each other in a cryptographically secure manner. Ths project is an implementation of blockchain technology and has the following features:

  1. Distributed Ledger
  2. Proof of Work Consensus
  3. Permissionless
  4. Cryptographically Secure

This project simulates a distributed environment by running nodes in different ports simultaniously.

To join from another node, open terminal and start server by passing new port number in the command line as >npm start 8050. Use sequential port numbers for new nodes.
